1st Exam

Real Number System & Inequalities

  1. Review of the Real Number System as an ordered field.
    The real numbers ℝ form an ordered field under + and ×. Key properties: Closure, Commutativity, Associativity, Distributivity, Identity elements (0 and 1), Inverses, and the Order axiom (trichotomy: for any a,b exactly one holds: a<b, a=b, or a>b).
  2. Is the integer 7 a solution to x + 2 < 10?
    Yes. Substitute: 7 + 2 = 9, and 9 < 10 ✓. So 7 is a solution.
  3. What are the solutions to −6 < x < 7? Show in a number line.
    All real numbers strictly between −6 and 7. In interval notation: (−6, 7). Number line: open circle at −6, open circle at 7, shaded between them.
  4. Write x > 10 in interval notation.
    (10, +∞) — open parenthesis at 10 (not included), extends to positive infinity.
  5. Is −8 a solution to −20 < x < 9?
    Yes. Check: −20 < −8 ✓ and −8 < 9 ✓. So −8 is in the interval.
  6. What is the distance between 15 and −8 on the number line?
    Distance = |15 − (−8)| = |15 + 8| = 23 units.
  7. Why does |x| = −6 have no solution?
    Absolute value is always non-negative (|x| ≥ 0 for all x). A negative right-hand side (−6) is impossible, so there is no solution.
  8. Why does |x| = 3 have 2 solutions?
    |x| = 3 means x is 3 units from 0. Two numbers satisfy this: x = 3 and x = −3.
  9. Why is |−6| = 6?
    Absolute value measures distance from 0, regardless of sign. −6 is 6 units from 0, so |−6| = 6.
  10. What is |10 − (−7 + 4)|?
    Simplify inside: −7 + 4 = −3. Then 10 − (−3) = 13. So |13| = 13.
  11. What are the solutions of |x + 5| = 2?
    Split: x + 5 = 2 → x = −3, or x + 5 = −2 → x = −7. Solutions: {−7, −3}.
  12. What are the solutions of −3x > 9?
    Divide both sides by −3 and flip the inequality: x < −3. Solution: (−∞, −3).
  13. What are the solutions of 5 ≥ x − 4 > 10?
    Add 4 to all parts: 9 ≥ x > 14. But 9 ≥ x and x > 14 is a contradiction (no number is both >14 and ≤9). No solution.

2D Coordinate System

  1. Coordinates of a point 6 units below the x-axis and 10 units to the right of the y-axis?
    Moving right means x is positive, below means y is negative. Answer: (10, −6).
  2. Vertical distance between (−10, 6) and (4, −9)?
    Vertical distance = |y₂ − y₁| = |−9 − 6| = |−15| = 15 units.
  3. Horizontal distance between (8, −3) and (5, 9)?
    Horizontal distance = |x₂ − x₁| = |5 − 8| = |−3| = 3 units.
  4. Distance of (18, −15) from the y-axis?
    Distance from the y-axis = |x| = |18| = 18 units.
  5. Coordinates of a point 8 units above x-axis and 30 units left of y-axis?
    Above x-axis: y = +8. Left of y-axis: x = −30. Answer: (−30, 8).
  6. In what quadrant if x > 0 and y = −3?
    x > 0 (positive x), y = −3 (negative y). That is Quadrant IV.
  7. Show any line with a y-intercept of −5.
    Any line crossing the y-axis at −5, e.g., y = x − 5 or y = 2x − 5. The y-intercept is the point (0, −5).
  8. Show the line with x-intercept 4 and y-intercept −3.
    Using intercept form: x/4 + y/(−3) = 1. Rearranged: 3x − 4y = 12. Passes through (4, 0) and (0, −3).

Lines

  1. Slope of the line through (−5, 3) and (4, −6)?
    m = (y₂−y₁)/(x₂−x₁) = (−6−3)/(4−(−5)) = −9/9 = −1.
  2. Slope of the line 3x − 6y = 8?
    Rewrite in slope-intercept form: −6y = −3x + 8 → y = (1/2)x − 4/3. Slope = 1/2.
  3. Find 3 points on 2x − 3y = 4.
    Let x=0: −3y=4, y=−4/3 → (0, −4/3). Let x=2: 4−3y=4, y=0 → (2, 0). Let x=5: 10−3y=4, y=2 → (5, 2).
  4. Line A ⊥ Line B. Slope of A = 3. Slope of B?
    Perpendicular slopes are negative reciprocals. Slope of B = −1/3.
  5. Draw the line through (−2, 5) with slope 2/3.
    Start at (−2, 5). Slope 2/3 means: rise 2, run 3. Next point: (−2+3, 5+2) = (1, 7). Plot both points and draw the line. Equation: y−5 = (2/3)(x+2) → y = (2/3)x + 19/3.
  6. Equation of the line with slope 1/4 and y-intercept −5?
    Using slope-intercept form y = mx + b: y = (1/4)x − 5.

Operations on Functions — f(x) = 3x−5, g(x) = x²+1

  1. f(x) + g(x) = ?
    (3x−5) + (x²+1) = x² + 3x − 4
  2. f(x) − g(x) = ?
    (3x−5) − (x²+1) = −x² + 3x − 6
  3. f(x) · g(x) = ?
    (3x−5)(x²+1) = 3x³+3x−5x²−5 = 3x³ − 5x² + 3x − 5
  4. f(x) / g(x) = ?
    (3x−5)/(x²+1). Domain: all reals since x²+1 ≠ 0 for any real x.
  5. Describe the domain of f(x) = 3x−5.
    f(x) is a linear function — no restrictions. Domain: all real numbers (−∞, +∞).
  6. f(x) = x² + 3x − 4. What is f(−3)?
    f(−3) = (−3)² + 3(−3) − 4 = 9 − 9 − 4 = −4
  7. f(x) = x²+3, g(x) = 3x−7. What is f(x) − g(x)?
    (x²+3) − (3x−7) = x² − 3x + 10
  8. f(x) = x²−2, g(x) = 3x+1. Find f(g(x)).
    f(g(x)) = f(3x+1) = (3x+1)² − 2 = 9x²+6x+1−2 = 9x² + 6x − 1

Test III — Problem Solving (6 pts)

Variable cost = Php 250/unit, fixed costs = Php 800,000, selling price = Php 400.

  1. (a) Write total cost C as a function of units produced.
    C(x) = 250x + 800,000 where x = number of units. Fixed cost is constant (800,000); variable cost grows with x.
  2. (b) Write profit P as a function of x.
    Revenue = 400x. Profit = Revenue − Cost.
    P(x) = 400x − (250x + 800,000) = 150x − 800,000
    Break-even when P=0: x = 800,000/150 ≈ 5,334 units.

Test II — Find the Derivative

Trig Derivatives (tan, cot, sec, csc)

  1. y = tan x + x
    d/dx[tan x] = sec²x, d/dx[x] = 1. y' = sec²x + 1
  2. f(x) = 3 sec x − cot x
    d/dx[3sec x] = 3sec x tan x, d/dx[−cot x] = csc²x. f'(x) = 3sec x tan x + csc²x
  3. g(x) = x² · tan x
    Product rule: (x²)'tan x + x²(tan x)' = 2x·tan x + x²·sec²x. g'(x) = 2x tan x + x² sec²x
  4. h(x) = csc x / x
    Quotient rule: [(csc x)'·x − csc x·1] / x² = [−csc x cot x · x − csc x] / x². h'(x) = −csc x (x cot x + 1) / x²

Chain Rule & General Power Rule

  1. y = (3x + 1)&sup5;
    Chain rule: 5(3x+1)&sup4; · (3x+1)'. Derivative of (3x+1) = 3. y' = 15(3x+1)&sup4;
  2. f(x) = √(x² + 4)
    Rewrite as (x²+4)1/2. Chain rule: (1/2)(x²+4)−1/2·2x. f'(x) = x / √(x²+4)
  3. g(x) = sin(4x)
    Chain rule: cos(4x) · 4. g'(x) = 4cos(4x)
  4. h(x) = cos(x³ − 2x)
    Chain rule: −sin(x³−2x) · (3x²−2). h'(x) = −(3x²−2)sin(x³−2x)

Mixed Rules

  1. y = 5x³ − 2√x + 9
    Power rule each term: d/dx[5x³]=15x², d/dx[−2x1/2]=−x−1/2=−1/√x, d/dx[9]=0. y' = 15x² − 1/√x
  2. y = (x² + 3) · cos x
    Product rule: (x²+3)'cos x + (x²+3)(cos x)' = 2x·cos x + (x²+3)(−sin x). y' = 2x cos x − (x²+3)sin x
  3. y = tan x / (1 + sec x)
    Quotient rule: [(1+sec x)(sec²x) − tan x(sec x tan x)] / (1+sec x)². Simplify using sec²x−tan²x=1: y' = sec x / (1 + sec x)
  4. y = sin(x² + 1)
    Chain rule: cos(x²+1) · 2x. y' = 2x cos(x²+1)
  5. y = (2x − 1)&sup4; · cos(3x)
    Product + chain: [(2x−1)&sup4;]'cos(3x) + (2x−1)&sup4;[cos(3x)]' = 4(2x−1)³·2·cos(3x) + (2x−1)&sup4;·(−sin(3x))·3. y' = 8(2x−1)³cos(3x) − 3(2x−1)&sup4;sin(3x)

Test III — Applications

1. Particle: s(t) = t³ − 6t² + 9t (meters, t in seconds)

  1. (a) Find the velocity function v(t).
    v(t) = s'(t) = 3t² − 12t + 9. v(t) = 3t² − 12t + 9
  2. (b) Velocity at t = 0, 2, and 4.
    v(0) = 9 — moving forward at 9 m/s. v(2) = 12−24+9 = −3 m/s (moving backward). v(4) = 48−48+9 = 9 m/s (moving forward again).
  3. (c) When is the particle at rest?
    Set v(t) = 0: 3t²−12t+9=0 → t²−4t+3=0 → (t−1)(t−3)=0. t = 1 s and t = 3 s.
  4. (d) Find the acceleration function a(t).
    a(t) = v'(t) = d/dt[3t²−12t+9]. a(t) = 6t − 12

2. Coffee shop: P = 4.00x − x²/10,000 − 6,000

  1. (a) Marginal profit dP/dx.
    dP/dx = d/dx[4x − x²/10000 − 6000] = 4 − x/5000
  2. (b) Number of cups that maximizes profit.
    Set dP/dx = 0: 4 − x/5000 = 0 → x = 20,000. Check: d²P/dx² = −1/5000 < 0 (maximum confirmed). x = 20,000 cups/day.
  3. (c) Maximum profit.
    P(20000) = 4(20000) − (20000)²/10000 − 6000 = 80000 − 40000 − 6000 = Php 34,000

3. Drone: s(t) = 2 + 6t − t²

  1. (a) Average velocity over [0, 4].
    Avg v = [s(4)−s(0)]/(4−0). s(4)=2+24−16=10, s(0)=2. Avg v = (10−2)/4 = 2 m/s
  2. (b) Average velocity over [4, 6].
    s(6)=2+36−36=2, s(4)=10. Avg v = (2−10)/(6−4) = −8/2 = −4 m/s (drone moving backward)

4. Rocket: s(t) = −16t² + 80t + 5

  1. (a) Find the velocity function v(t).
    v(t) = s'(t) = −32t + 80
  2. (b) Instantaneous velocity at t = 2s.
    v(2) = −32(2) + 80 = −64 + 80 = 16 ft/s (still moving upward)
  3. (c) Instantaneous velocity at t = 5s — what does it mean?
    v(5) = −32(5) + 80 = −160 + 80 = −80 ft/s. Negative means the rocket is moving downward at 80 ft/s at t=5s. (Note: the rocket reaches its peak when v=0: t = 80/32 = 2.5s.)
